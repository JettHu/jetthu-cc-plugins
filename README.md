# jetthu-cc-plugins

Personal Claude Code plugin marketplace by JettHu.

## Plugins

| Plugin | Description |
|--------|-------------|
| [x-review](https://github.com/JettHu/cc-codex-cross-review) | Cross-model plan review: adversarial feedback loop with session reuse. Works best with [codex-plugin-cc](https://github.com/openai/codex-plugin-cc) for automated Codex delegation, or use any LLM as reviewer manually |

## Install

```bash
# Add this marketplace
claude plugins marketplace add JettHu/jetthu-cc-plugins

# Install a plugin
claude plugins install x-review
```
